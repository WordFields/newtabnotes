Markdown auto format

Use markdown-like syntax to seamlessly apply formatting as you type, for example:

  • Type ** or __ on either side of your text to add **bold** mark.
  • Type * or _ on either side of your text to add *italic* mark.
  • Type ` on either side of your text to add `inline code mark`.
  • Type ~~ on either side of your text to add ~~strikethrough~~ mark.
  • Type *, -  or  + and then hit space to create a bulleted list.
  • Type 1.  or  1) and then hit space to create a numbered list.
  • Type > and then hit space to create a block quote.
  • Type # and then hit space to create an H1 heading.
  • Type ## and then hit space to create an H2 sub-heading.
  • Type ### and then hit space to create an H3 sub-heading, etc...
  • Type --- to create a horizontal rule.
  • Type ``` to create a code block.

Keyboard shortcuts

  • Format text as bold Ctrl+B, italic Ctrl+I, underline Ctrl+U, strike Ctrl+Shift+X
  • Highlight text with Ctrl+Shift+H or make it embossed with Ctrl+Shift+K.
  • Just type an URL like https://newtabnotes.com and hit space to add a link, or use Ctrl+K.
  • Use standard Ctrl+Z for undo and Ctrl+Y for redo.
  • For emojis hit Win+Period.
